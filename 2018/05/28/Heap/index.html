Binary Heap

Properties:

  1. It’s a complet tree (All levels are completely filled except possibly the last level and last level has all keys as left as possible)
  2. A Binary Heap is either Min Heap or Max Heap.

Binary Heap is typically represented as array.

Representation Explanation
Arr[(2*i)+1] Returns the left child node
Arr[(2*i)+2] Returns the right child node
Arr[(i-1)/2] Returns the parent node

Application of Heaps

  1. Heap Sort: O(nLogn) time
  2. Priority Queue
  3. Graph Algorithms

Some Classic Problems

  • K’th Largest Element in an array
  • Sort an almost sorted array
  • Merge K Sorted Arrays

leetcode - 215 Kth Largest Element in an Array

C++ build-in data structure priority queue

priority_queue<Type, Container, Functional> p

p.push() add element

p.top() get the top element of the tree

p.pop() pop out the top element
